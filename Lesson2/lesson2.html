Задание 1

Почему код даёт такие результаты:

var a = 1, b = 1, c, d; c = ++a; alert(c);
Результат = 2, потому что ++а это префиксная запись. К "а" сначала делается инкремент, а потом идёт присваивание;

d = b++; alert(d);
Результат = 1, т.к. запись постфиксная и сначала идёт присваивание, а потом инкремент. Т.е. на момент "alert" b = 2;

c = (2+ ++a); alert(c);
Результат = 5, т.к. запись префиксная и на момент прибавления 2 "а" = 3;

d = (2+ b++); alert(d);
Результат = 4, т.к. запись постфиксная. На момент операции "b" = 2, а после сложения к ней добавляется инкремент, поэтому в момент "alert" b = 3

alert(a); // 3. логичный итог всех вышеприведенных операций
alert(b); // 3. логичный итог всех вышеприведенных операций
